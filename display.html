Dispaly inline

- afiseaza elementele in rand, nu pot fi stilizate folosind width sau height, margin sau padding

- cand lucram cu elemente cu display inline NU folosim stilizari de tip: width, height, padding

- elemente care au by default display inline: a, label, input


Display block

- afiseaza elementele in coloana (unul sub celalalt)

- pot fi stilizate cu width, height, margin, padding

- elemente care au by default display block: p, div, h1-h6

Display inline-block

- afiseaza elementele in rand

- POT fi stilizate cu width, height, margin, padding
